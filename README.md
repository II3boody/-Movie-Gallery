# 🎬 Movie Gallery

A responsive **Movie Gallery web app** that allows users to browse
movies, search by title, filter by genre, sort by year or rating, and
save favorite movies using localStorage.

## 🚀 Features

- 🔍 **Search Movies** by title
- 🎭 **Filter by Genre**
- 📊 **Sort Movies**
  - Newest
  - Oldest
  - Highest Rating
- ❤️ **Favorite Movies**
  - Save favorites using `localStorage`
- 🔲 **Grid / List View Toggle**
- 📱 **Responsive Design**
- ⚡ Smooth animations and modern UI

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **LocalStorage**
